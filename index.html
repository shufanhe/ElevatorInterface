Elevator Interface

This project examines the user interface of an elevator inside a university building. After user obeservations and interview, we create two archetypal user personas and a story board demonstrating a user journey.

The Interface:

Many elevators on our university campus require card access to get to certain floors, all time or after certain hours. This prevents students looking for study spaces to accidentally wander into an office or a meeting room, and protects public and private property stored in certain locations. To access those floors, a user needs to swipe their campus ID card first, then press the button to the corresponding floor. The following sketch is an example from the library elevator.
